# 🌌 VR 太阳系漫游 · Solar System

一个基于 **Three.js + WebXR** 的太阳系模拟与漫游项目。
桌面端用鼠标即可自由漫游；接上 VR 头显（或手机 Cardboard）后，点 **Enter VR** 即可进入沉浸式宇宙。

无需构建、无需安装依赖、**完全离线可用**（Three.js 已打包进 `vendor/`）。

---

## ✨ 功能

- **太阳 + 八大行星**：按"可观赏比例"缩放，保留相对大小与轨道顺序；太阳带颗粒纹理与光晕。
- **程序化真实纹理**：木星横向条纹 + 大红斑、地球海陆/沙漠/冰盖 + 独立飘动的云层、火星锈红 + 极冠、天王星/海王星冰蓝色带——全部用 Canvas 程序生成，无需外部图片。
- **真实自转轴倾角**：地球 23.4°、天王星 97.8°（横躺打滚）、金星 177°（近乎倒转）等均按真实值。
- **公转 + 自转**：各天体按相对真实周期运行（含金星/天王星的逆向自转），可见轨道线。
- **卫星与行星环**：地球带月球；土星环含卡西尼缝，土星/天王星环随自转轴一起倾斜。
- **小行星带**：火星与木星之间 1500 颗小行星缓慢公转。
- **星空背景**：6000 颗星包裹整个场景。
- **自由漫游**：鼠标拖拽旋转、滚轮缩放、右键平移。
- **点击聚焦**：点行星或左下角名称按钮，相机平滑跟随并弹出科普信息卡。
- **时间控制**：暂停/继续、0–500× 调速。
- **WebXR**：右下角 `Enter VR` 进入沉浸模式（需支持 WebXR 的浏览器 + 头显）。

## 🚀 运行方法

因为用到了 ES Module 和 importmap，**不能直接双击打开 `index.html`**（`file://` 会被浏览器的跨域策略拦住），需要用一个本地静态服务器。任选一种：

```bash
# 方式一：Python（几乎人人都有）
python3 -m http.server 8000

# 方式二：Node
npx serve .
```

然后浏览器打开 <http://localhost:8000> 即可。

> 推荐用 Chrome / Edge / Firefox 等现代浏览器。进入 VR 需要支持 WebXR 的浏览器与设备。

## 🛠 技术说明

- **比例不是真实比例**：真实尺度下行星相对太阳与轨道太小，肉眼几乎不可见。本项目对半径与轨道距离做了非线性压缩，**保留顺序与相对大小关系**，优先保证可观赏性。
- **光照**：太阳处放置点光源，并刻意关闭了距离衰减（`decay=0`），让最远的海王星也能被照亮。
- **行星外观**：使用 Canvas 程序化生成的等距柱状贴图（木星条纹、地球海陆云层、土星环卡西尼缝等），完全离线、无需下载 NASA 贴图。后续若需更高保真度，可平滑替换为真实卫星影像。
- **自转轴倾角**：通过「公转 pivot → 竖直 holder → 倾斜 tiltGroup → 自转 mesh」的层级实现，行星环与卫星挂在 tiltGroup 上，随赤道面一起倾斜。
