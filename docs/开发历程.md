# 虚拟看房 · 开发历程 (Build Log)

> 本文档按时间线记录「虚拟看房 / Virtual House」这个项目，是如何通过与 **Claude Code** 结对开发、一轮一轮迭代出来的。
> 每一条都包含：**需求**（我提的）→ **实现**（做了什么）→ **提交**（git commit）。
>
> 在线体验：<https://1739467001-svg.github.io/Virtual-Reality/> ·
> 仓库：<https://github.com/1739467001-svg/Virtual-Reality>

---

## 这是什么

一个**纯浏览器、零构建**的 3D 虚拟看房 / 装修 Demo：第一人称走进房间，自由摆放家具、换装软装、调灯光天气四季、切换户型，支持手机 / 平板 / 电脑 / VR 头显。

**技术栈**：Three.js r160（CDN + import map，无打包）、原生 HTML/CSS/ES Modules、Canvas 程序化纹理、少量 Khronos CC 许可的 glTF 模型、GitHub Actions + Pages 自动部署。

**协作方式**：我用自然语言提需求，Claude Code 负责设计、写代码、跑测试、提交并部署，每轮结束我在真机上验收、再提下一轮。

---

## 方法论（贯穿全程）

- **小步快跑**：每个需求 = 一轮 = 一次提交 + 一次部署，始终保持可运行、可回退。
- **冒烟测试护栏**：`test/smoke.mjs` 用 Node 无头加载场景、跑一遍所有交互逻辑，抓运行时错误；后期接入 CI 作为**部署前门禁**。
- **"盲做"+ 真机验收**：AI 看不到画面，所有视觉/手感靠逻辑与几何校验（如门洞可通行宽度、碰撞盒不重叠），最终由我在真机上验收视觉与体验。
- **渐进增强 + 开关**：高风险/视觉类改动（后期处理、户型、真实模型）都做成**可开关 / 可切换 / 优雅降级**，避免破坏既有体验。

### 第 1 阶段 · 画质升级 + 部署修复（2026-06-15）

- **需求**：画质升级（真实模型）。
  **实现**：接入后期处理管线（`EffectComposer`：UnrealBloom 辉光 + SMAA 抗锯齿，面板「高画质 HQ」开关，VR 下自动旁路）；内置真实 glTF 模型（真皮沙发 / 锦缎单椅，Khronos CC，本地化避免跨域）。 `075319c`
- **意外发现并修复**：GitHub Pages 一直在构建一个**过期的 `gh-pages` 分支**，导致此前几次推送根本没上线。改为 **GitHub Actions 部署仓库根目录**，每次推 `main` 自动发布。 `4f7b829`
- **需求**：旋转家具 + 拍照。
  **实现**：移动模式下 Q/E 或 ⟲⟳ 旋转（朝向写入分享链接）；「📸 拍照」导出当前视角 PNG。 `1f40a2a`
- **需求**：更新仓库说明与 README。
  **实现**：修正了**严重过期**的仓库简介（之前还写着"太阳系 VR"）、topics、README（部署说明、结构树、技术说明全部对齐现状）。 `6080783`

---

### 第 2 阶段 · 多房间户型（2026-06-15）

- **需求**：做多房间户型（此前几套本质是同一空间在变）。
  **实现**：新增「户型」开关，在客厅之后追加卧室，复用原入户门作为房间间门洞；玩家碰撞扩展到隔墙，小地图按户型自适应。 `3f9be92`
- **需求**：卧室采光 + 卫生间户型。
  **实现**：卧室开窗（含窗外绿植）；新增「一室一厅一卫」——嵌入式卫生间（隔墙+门、马桶、带镜洗手台、瓷砖地）。 `1998c78`
- **需求**：加淋浴/浴缸、厨房区。
  **实现**：开放式厨房（地柜+台面、吊柜、水槽、四眼灶+抽油烟机、冰箱）；卫生间玻璃淋浴房。 `f9871be`
- **需求**：再加户型，至少 3 套且要真正不同。
  **实现**：新增「两室一厅」——中央隔墙分主卧+次卧，各带窗、门洞相通。 `07d6613`
- 几何全程用无头脚本校验：门洞可通行宽度（如客厅↔卧室 0.52m、卫生间门 0.39m）、洁具/床不越界、不挡门。

### 第 3 阶段 · 工具与工程化（2026-06-15）

- **需求**：测量工具 + 设计深化。
  **实现**：测量模式（准星点两点得直线距离，3D 连线+浮标）；选中家具**换色 / 缩放**（写入分享链接）。 `1809cf6`
- **需求**：内容补全（再扩真实 glTF、可换挂画、厨房换色）。
  **实现**：真实玻璃花瓶 glTF；程序化「床 / 餐桌椅」（床与餐桌椅在可靠的 CC 源里无现成模型，坦诚用程序化）；可换挂画（初版 3 幅）；厨房柜体/台面 4 套配色。 `badab1f`
- **需求**：工程门禁 + 体验项。
  **实现**：**CI 部署前先跑冒烟测试**（不过则拦截）；模型加载提示；首次操作引导 + 帮助按钮；**净尺寸标注**（各房间长宽面积）；分享链接**二维码**（内置离线 qrcode.js）。 `9ca4483`
- **需求**：继续（工程健康）。
  **实现**：提交 `package-lock.json`、CI 改用 `npm ci`；真实灯具（9MB Lantern）**懒加载**——只在首次放入时下载，不拖慢首屏。 `18d1ceb`

### 第 4 阶段 · 氛围：天空 / 天气 / 四季 / 昼夜（2026-06-15 → 06-16）

- **需求**：窗外天空盒 + 昼夜平滑过渡 + 多场景天气 + 春夏秋冬。
  **实现**：新增 `environment.js`——渐变天空盒背景；昼/昏/夜平滑补间；晴/雨/雪（窗外粒子，阴天调灰加雾）；四季切换窗外植被地面色。 `3bdd520`
- **需求**：室内随天气/季节微调 + 天空加日月星。
  **实现**：雨雪降低室内环境光/曝光（更有氛围）；太阳/月亮辉光 + 700 颗星空夜晚淡入；阴天遮蔽日月星。 `4617307`
- **需求**：日月真实弧线移动（时间连续滑动）+ 窗外远景升级。
  **实现**：时间改为 **0–24h 连续滑杆**，按 24h 关键帧插值，太阳/月亮沿真实弧线升落；窗外加**城市天际线（夜晚亮灯）+ 远山**。 `6ce8e62`

---

### 第 5 阶段 · 摆放体验深化（2026-06-16）

- **需求**：把每间房的固有家具也做成可拾取。
  **实现**：床/床头柜/衣柜/马桶/洗手台/淋浴房从静态装饰改为**可拾取实体**（迁入家具目录，按户型投放），统一支持移动/旋转/换色/缩放/删除/分享。 `42107ba`
- **需求**：拿起东西时无法转身左右、只能前后。
  **修复**：搬运逻辑从"贴地准星点"改为**正前方固定距离悬浮**——转身即左右扫动，任意视角都灵敏；并修了夹取范围被限制在客厅的 bug。同时给各房补静态点缀。 `0352090`
- **需求**：加下蹲（C 键）看桌下/床品，比跳跃更贴合看房。
  **实现**：C 键 / 面板按钮平滑切换站立↔下蹲（1.65↔0.85m）。 `0398338`
- **需求**：蹲姿更真实。
  **实现**：下蹲时移速减半 + 视角轻微下倾约 7.5°。 `2c6c20d`

### 第 6 阶段 · 细节与摆件（2026-06-16）

- **需求**：每间房再加细节。
  **实现**：厨房（果盘/调料瓶/刀架/水壶/吊灯）、卫生间（毛巾/地垫/置物架/绿植）、卧室（床头挂画/角落绿植）、客厅（门垫/大绿植）。 `69a9c0d`
- **需求**：部分点缀可拾取 + 墙面装饰画可换。
  **实现**：绿植/花瓶变为可拾取实体；挂画扩展到 **6 幅画作**，客厅+卧室一起切换。 `7b63587`
- **需求**：更多小摆件（书/相框/地球仪等）。
  **实现**：新增可拾取小摆件——书籍、相框、地球仪、蜡烛。 `a664ec5`
- **需求**：搬运可调高度 / 自动吸附台面。
  **实现**：拾起物件**向下检测表面自动吸附**（地板/桌/床/书架），R/F/滚轮/按钮微调高度，放下保持原位，高度写入分享链接。 `f8e1e2f`
- **需求**：叠放检测优化（优先吸附最近台面）。
  **实现**：落点改为"瞄准高度之下最近的台面"——默认吸顶层（自动上桌/叠物），降低瞄准可放到下层（如茶几下的地面）。 `d94c55a`

### 第 7 阶段 · 交互反馈与性能（2026-06-16）

- **需求**：选中/悬停高亮 + 重叠红色警告 + 精简投影灯。
  **实现**：移动模式下悬停**黄框** / 手持**绿框** / 重叠墙或家具时**红框**；投影灯从 ~5 盏精简到 **2 盏**（仅太阳+吸顶），移动端更顺。 `33decf4`

### 第 8 阶段 · 多端自适应（2026-06-16）

- **需求**：适配手机 / 平板 / 电脑，面板可折叠。
  **实现**：控制面板**分区折叠手风琴**（点标题展开/收起）；响应式断点（平板收窄、手机自适应）。 `43ddd24`
- **需求**：手机端把面板改成底部抽屉/标签（更像 App）。
  **实现**：手机上面板变 **App 式底部抽屉**——右下「☰ 菜单」唤出，顶部标签切分区，遮罩/把手收起，平时不挡画面与摇杆。 `034f71e`
